第一步: 打开GitHub主页, 登录账号
https://github.com/

第二步: 点击左上角绿色的New按钮, 创建一个新的仓库

第三步: 给仓库起一个名字, 比如说 NoteBook, 
可以勾选上README, 这样就会自动创建一个README.md文件
然后点击Create Repository按钮

第四步: 安装Github桌面版
https://desktop.github.com/



1. GitHub Theme: 界面主题
2. Fluent Icons: 图标主题
3. vscode-icons: 文件图标
4. Markdown All in One: Markdown预览补全
5. GitHub Copilot: 代码补全
6. GitHub Copilot Chat: 内置聊天AI
7. C/C++: C/C++语言支持
8. Python: Python语言支持
9. Pylance: Python语言支持
10. Python Debugger: Python调试器
11. Black Formatter: Python代码格式化
12. TabOut: Tab跳出括号
13. Code Runner: 代码运行自动化
