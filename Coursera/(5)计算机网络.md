# 构造可靠数据传输协议

## 完全可靠信道的可靠数据传输: rdt1.0

![](https://img.ethancao.cn/202410151000433.png =600x)


| Func                      | Desc             |
| ------------------------- | ---------------- |
|                           | - 发送端 -       |
| `rdt_send(data)`          | 从高层承接数据   |
| `packet = make_pkt(data)` | 将数据封装为分组 |
| `udt_send(packet)`        | 往信道发送分组   |
|                           | - 接收端 -       |
| `rdt_rcv(packet)`         | 从信道接收分组   |
| `data = extract(packet)`  | 将分组提取为数据 |
| `deliver_data(data)`      | 向高层传递数据   |

## 比特差错信道的可靠数据传输: rdt2.0

![](https://img.ethancao.cn/202410151008279.png =600x)

> 缺陷: 未考虑ACK或NAK分组传输受损的情况

# 流水线可靠数据传输协议

## GBN协议

### 发送方

```sh
{
    base = 1        # 初始化基序号
    nextseqnum = 1  # 初始化待发送序号
}
{
    # 从高层承接数据
    rdt_send(data)
    # 如果待发送序号在窗口内
    if (nextseqnum < base + N)
        # 将(序号,数据,校验和)封装为分组并发送
        sndpkt = make_pkt(nextseqnum,data,checksum)
        udt_send(sndpkt)
        # 如果是窗口首个分组, 则启动定时器
        if (base == nextseqnum)
            start_timer()
        # 继续处理下个序号
        nextseqnum += 1
    else
        # 否则拒绝发送
        refuse_send()
}
{
    # 如果超时
    time_out()
    start_timer()
    # 则重传窗口内所有分组
    udt_send(sndpkt[base])
    udt_send(sndpkt[base+1])
    ......
    udt_send(sndpkt[nextseqnum-1])
}
{
    # 如果分组校验和错误, 则直接忽略
    rdt_rcv(rcvpkt) && corrupt(rcvpkt)
}
{
    # 如果分组校验和正确, 则右移窗口基序号
    rdt_rcv(rcvpkt) && notcorrupt(rcvpkt)
    base = getacknum(rcvpkt) + 1
    # 如果窗口接收完毕, 则停止定时器
    if (base == nextseqnum)
        stop_timer()
    # 如果窗口还在接收, 则重置定时器
    else
        start_timer()
}
```


### 接收方

```sh
{
    expected_seqnum = 1                 # 初始化期望序号
    sndpkt = make_pkt(0,ACK,checksum)   # 初始化ACK-0分组
    udt_send(sndpkt)                    # 发送ACK-0分组
}
{
    rdt_rcv(rcvpkt)                             # 接收到分组
        && notcorrupt(rcvpkt)                   # 校验和正确
        && has_seqnum(rcvpkt,expected_seqnum)   # 期望序号匹配
    # 反馈并更新 ACK-expected_seqnum
    sndpkt = make_pkt(expected_seqnum,ACK,checksum)
    expected_seqnum += 1
    udt_send(sndpkt)
    # 提取数据并向高层传递
    data = extract(rcvpkt)
    deliver_data(data)
}
```
