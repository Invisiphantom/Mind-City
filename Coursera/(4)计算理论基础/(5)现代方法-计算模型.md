## 图灵机的定义

图灵机的定义(Turing Machine)
<只读输入带 工作带 输出带>
3元组$(\Gamma,Q,\delta)$称为图灵机
1. 字母表: 有限集合$\Gamma$
2. 状态集: 有限集合$Q$ 起始状态$q_{start}$ 停机状态$q_{halt}$
3. 转移函数: $\delta:Q\times\Gamma^{k}\to Q\times\Gamma^{k-1}\times\{L,S,R\}^k$

示例: 对于任意$x\in\{0,1\}^*$, 如果x是回文则PAL(x)=1, 否则PAL(x)=0 
构造图灵机M有3条带(输入带 工作带 输出带)和字母表$\{\rhd,\square,0,1\}$
1. 将输入带复制到工作带
2. 将输入带带头移回最左侧
3. 输入带带头向右移动, 工作带带头向左移动$$
    如果发现了值不相等, 则停机并输出0 
4. 正常结束, 停机并输出1

## 图灵机运行时间

图灵机运行时间的定义
<最弱模型来模拟最强模型 只导致时间按平方增长>
已知布尔函数$f:\{0,1\}^*\to\{0,1\}$和时间函数$T:\mathbb{N}\to\mathbb{N}$
如果图灵机对于任意输入$x\in\{0,1\}^*$, 停机时会将$f(x)$写在它的输出带
且最多只需要执行$T(|x|)$步, 则称M在$T(n)$时间内计算$f$

<br>

限定字母表后的时间增量(Limited Alphabet)
已知算法函数$f:\{0,1\}^*\to\{0,1\}$和时间函数$T:\mathbb{N}\to\mathbb{N}$
如果字母表$\Gamma$上的图灵机$M$可在时间$T(n)$内进行计算
那么也能被字母表$\{0,1,\square,\rhd\}$上的图灵机$\bar M$
在时间$3\log|\Gamma|T(n)$内计算
证明: 将$\Gamma$中的每个符号用$\log|\Gamma|$位进行编码
1. 让每个带头右移$\log|\Gamma|$进行读值, 存储到状态编码中
2. 依照M的转移函数进行状态跳转
3. 让每个带头左移$\log|\Gamma|$进行写值
4. 让每个带头按需移动$\log|\Gamma|$

<br>

限定单带后的时间增量(Single Tape)
已知算法函数$f:\{0,1\}^*\to\{0,1\}$和时间函数$T:\mathbb{N}\to\mathbb{N}$
如果k带图灵机$M$可在时间$T(n)$内进行计算
那么也能被单带图灵机$\bar M$在时间$O(kT^2(n))$内计算

证明: 构造模拟带, 其为k条纸带的交替拼接, ^用于标记读写头
将第一条带编码为位置$1,k+1,2k+1,3k+1,4k+1,\cdots$
将第二条带编码为位置$2,k+2,2k+2,3k+2,4k+2,\cdots$
k带转移函数: $\delta:Q\times\Gamma^k\to Q\times\Gamma^k\times\{L,R,S\}^k$
|        |                  |                  |                  |          |
| ------ | ---------------- | ---------------- | ---------------- | -------- |
| 纸带a  | $a_1$            | $a_2$            | $a_3$            | $\cdots$ |
| 纸带b  | $b_1$            | $b_2$            | $b_3$            | $\cdots$ |
| 纸带c  | $c_1$            | $c_2$            | $c_3$            | $\cdots$ |
| 模拟带 | $a_1b_1\hat c_1$ | $a_2\hat b_2c_2$ | $\hat a_3b_3c_3$ | $\cdots$ |

1. 从左到右扫描模拟带, 读取并记录所有带^的字符
2. 根据k带的转移函数, 来决定下一次每个带的状态
3. 从右到左扫描模拟带, 更新所有带^的字符

(1)扫描耗时为纸带长度$kT(n)$
(3)扫描耗时为$kT(n)$, 更新^标记耗时为$k*k$
三阶段共耗时$O[kT(n)]+O[kT(n)+k^2]=O(kT(n))$
执行$T(n)$回合, 总计时间为$O(kT(n))\times O(T(n))=O(kT^2(n))$

<br>

限定单向后的时间增量(Single Direction)
已知算法函数$f:\{0,1\}^*\to\{0,1\}$和时间函数$T:\mathbb{N}\to\mathbb{N}$
如果双向图灵机$M$可在时间$T(n)$内进行计算
那么也能被单向图灵机$\bar M$在时间$O(T(n))$内计算

证明: 构造模拟带, 其为双向纸带的正负交替拼接
|          |          |       |       |       |       |       |          |
| -------- | :------: | :---: | :---: | :---: | :---: | :---: | :------: |
| 双向纸带 | $\cdots$ | $-2$  | $-1$  |  $0$  |  $1$  |  $2$  | $\cdots$ |
| 模拟带   |   $0$    | $-1$  |  $1$  | $-2$  |  $2$  | $-3$  | $\cdots$ |

对于模拟带, 状态更新时需要左移或右移两位
故总计时间为$O(2T(n))=O(T(n))$

<br>

随机访存图灵机的定义(Random Access Turing Machine)
8元组$M=(Q,\Sigma,\Gamma,q_0,q_{accept},q_{reject},Q_{access},\delta)$称为随机访存图灵机
纸带字母表: 读写操作字符$R,W\in\Gamma$
转移函数: $\delta:Q\times\Gamma^{k+1}\to Q\times\Gamma^{k+1}\times\{L,R,S\}^k$
有访存状态集$Q_{access}\subseteq Q$, 如果处于$q\in Q_{access}$则执行
1. 若当前内存带为$011R$, 则从内存$Mem[011_{2}]$处读取数据到地址带
2. 若当前内存带为$011W\sigma$, 则向内存$Mem[011_{2}]$处写入数据$\sigma$

|         |       |       |       |       |          |
| ------- | ----- | ----- | ----- | ----- | -------- |
| 工作带1 | $a_1$ | $a_2$ | $a_3$ | $a_4$ | $\cdots$ |
| 工作带2 | $b_1$ | $b_2$ | $b_3$ | $b_4$ | $\cdots$ |

|        |     |     |     |     |                   |     |          |          |
| ------ | --- | --- | --- | --- | ----------------- | --- | -------- | -------- |
| 索引   | 0   | 1   | 2   | 3   | 4                 | 5   | 6        | $\cdots$ |
| 内存带 |     |     |     |     |                   |     | $\sigma$ | $\cdots$ |
| 地址带 | 0   | 1   | 1   | R   | $\sigma^\swarrow$ |     |          |          |
| 地址带 | 0   | 1   | 1   | W   | $\sigma^\nearrow$ |     |          |          |

<br>

限定无内存后的时间增量(None Memory)
已知算法函数$f:\{0,1\}^*\to\{0,1\}$和时间函数$T:\mathbb{N}\to\mathbb{N}$
如果RAM图灵机$M$可在时间$T(n)$内进行计算
那么也能被*多带*图灵机$\bar M$在时间$O(T^3(n))$内计算
特别地, 若地址长度为$O(1)$, 那么时间缩短为$O(T^2(n))$


证明: 对于多带图灵机, 构造内存模拟带来放访存操作对$(i,R[i])$
当RATM处于$q\in Q_access$时, 执行以下操作
1. 扫描内存模拟带, 寻找具有匹配地址的访存操作对
2. 若不存在匹配地址的操纵对, 则新增操作对$(i,R[i])$
3. 相应地对该访存操作对进行读写

操作对的个数$\leq T(n)$
操作对的长度$\leq T(n)$
故模拟内存带的长度$\leq T^2(n)$
所以步骤(1)需要耗时$O(T^2(n))$
执行$T(n)$回合, 总计时间为$O(T^3(n))$


## 图灵机的位串表示

图灵机的位串表示(Machine As Strings)
由于字母表$\Gamma$和状态集$Q$可直接从转移函数得到
故只需要对转移函数$\delta$进行编码

若$|\Gamma|\leq 32,|Q|\leq 4$, 则可用5位串对字符进行编码, 
用2位串对状态进行编码, 用2位串对带头的移动进行编码
引入分隔符号$-$, 得到该法则下的编码为

转移规则: $\langle (q_1)\times(a,b,c)\rangle \to\langle (q_2)\times(d,e,f)\times(L,S,R)\rangle$
位串表示: $\epsilon=$ 01-00001-00010-00011--10-00100-00101-00110-10-00-01
|          |       |                   |       |       |                   |           |
| -------- | ----- | ----------------- | ----- | ----- | ----------------- | --------- |
| 转移规则 | $q_1$ | $(a,b,c)$         | $\to$ | $q_2$ | $(d,e,f)$         | $(L,S,R)$ |
| 位串表示 | 01    | 00001-00010-00011 | --    | 10    | 00100-00101-00110 | 10-00-01  |

将所有的转移规则拼接即可得到$M=\epsilon_1:\epsilon_2:\cdots:\epsilon_n$
最后再将这些特殊字符映射到01串: $\langle 0,1,-,:\rangle\to\langle 00,01,10,11\rangle$
图灵机的位串编码满足的性质
1. $\{0,1\}^*$中的每个串均表示一个图灵机
    特别地, 将非法编码映射为空图灵机
2. 每个图灵机都存在无限种编码
    即允许在编码后插入任意个无意义的1


由于每个图灵机的对应位串是有限长的
所以图灵机的总数是可数的
$\{\epsilon,0,1,00,01,10,11,000,\cdots\}$

由于每个语言所对应的特征序列是无限长的
所以语言的总数是不可数的(对角线方法)
故存在不能被任何图灵机识别的语言

## 通用图灵机

通用图灵机的定义(Universal Turing Machine)
<能模拟图灵机运行过程的图灵机>
通用图灵机$U(a,x)=M_a(x)$, 其中图灵机编码a和输入字符串x

高效通用图灵机(Efficient Universal Turing Machine)
<能够快速模拟任意图灵机的执行过程>
如果$M_a$在输入x上至多运行T步后停机,
那么可以构造高效$U(a,x)$, 使得能够模拟$M_a$并在$O(T\log T)$步后停机

![](https://pic1.zhimg.com/80/v2-ee219c44836729229d76faa2541a30e8_1440w.webp =400x)
![](https://pic4.zhimg.com/80/v2-eb6a0e08d22554096c14bdec30423553_1440w.webp =400x)
首先证明宽松形式$O(T^2)$:
构造五条纸带: 输入带 主工作带 模型工作带 状态工作带 输出带
1. 输入带: 图灵机编码a 输入字符串x
2. 主工作带: 用元组形式模拟M的k-2条工作带
    当M的k-2个带头分别独立移动时, 主工作带采用"带头不动带子动",
    把每层都按相应方向进行整层移动
3. 模型工作带: 存放M的所有转移函数规则
4. 状态工作带: 存放M的对应当前状态

读写纸带元组: $O(1)$
扫描转移函数表: $O(|a|)$
移动整层纸带: $O(T)$
执行T回合的总时间: $O(T^2)$

![](https://pic1.zhimg.com/80/v2-262f6c654aead52a4816e625f00ea848_1440w.webp =400x)
![](https://pic4.zhimg.com/80/v2-c304963f5654cc592bffce298db95ea3_1440w.webp =400x)
然后证明高效形式$O(T\log T)$:
为避免每次都移动整层, 引入缓冲符号$\boxtimes$
|          |       |       |       |       |
| -------- | ----- | ----- | ----- | ----- |
| 右侧区间 | $R_0$ | $R_1$ | $R_2$ | $R_3$ |
| 左侧区间 | $L_0$ | $L_1$ | $L_2$ | $L_3$ |
| 区间宽度 | $2^1$ | $2^2$ | $2^3$ | $2^4$ |

约束条件:
1. 每个区间要么时全满(无缓冲), 要么是半满(半缓冲), 要么是全空(全缓冲)
2. 区间$R_i$与$L_i$互补, 即要么$R_i$满$L_i$空, 要么$R_i$空$L_i$满, 要么都是半满
3. 0号位的数据永远不会是缓冲符号$\boxtimes$


当该层进行左移操作时:
1. 将0号位移动到草稿带
2. 向右扫描, 直到遇见非空区间$R_i$
3. 将$R_i$的一半数据移动到$0,R_0,\cdots,R_{i-1}$
    使得这些空区间成为半满($1+1+2+\cdots+2^{i-1}=2^i$)
4. 由约束条件可知, 当前$L_{i-1},\cdots,L_0$为全满, $L_i$为半满或全空
    故可将$L_{i-1},\cdots,L_0$中的数据依次对应左移到$L_i,\cdots,L_1$中
    最后再将0号位从草稿带移动到 空出来的$L_0$中

考虑最坏情况: 某次左移遇见$R_i$, 使得$R_0-R_{i-1}$成为半满
如果带子此后一直左移, 要移动$2^0+\cdots+2^{i-1}=2^i-1$次将半满清空后
再进行一次左移才能再次遇见$R_i$, 故两次遇见$R_i$的移动至少要间隔$2^i$

由于带子至多移动T次, 那么对区间$R_i$的移动不超过$\frac{T}{2^i}$次
每次移动区间$R_i$需要耗时$O(2^i)$(右侧-草稿带-右侧 左侧-草稿带-左侧)
最多有$\log T$个区间, 故总时间为$O[\sum_i^{\log T}\frac{T}{2^i}O(2^i)]=O(T\log T)$


## 不可计算性

存在不能被任何图灵机计算的函数
证明: 布尔函数不可数, 而图灵机只有可数个, 故存在不可计算函数

丘奇-图灵命题(Church-Turing Thesis)
任何可被物理实现的计算装置, 均可被图灵机模拟

强丘奇-图灵命题(Strong Church-Turing Thesis)
任何可被物理实现的计算装置, 均可被图灵机在多项式时间内模拟


## 多项式复杂性类

DTIME记号的定义(Decidable Time Complexity Class)
已知时间函数$T:\mathbb{N}\to\mathbb{N}$和语言$L\subseteq\{0,1\}^*$
称语言$L\in D_{time}(T)$, 当且仅当存在运行时间为$O(T)$的关于L的判定器

非确定性图灵机的定义(Non-deterministic Turing Machine)
有两个转移函数$\delta_0,\delta_1$和特殊状态$q_{accept}$
每个步骤中都可任意选用两个转移函数之一(类似于穷举$\{0,1\}^*$)
如果**存在某个**转移函数的选用序列, 最终进入状态$q_{accept}$, 则接受输入

NDTIME记号的定义
已知时间函数$T:\mathbb{N}\to\mathbb{N}$和语言$L\subseteq\{0,1\}^*$
称语言$L\in ND_{time}(T)$, 当且仅当存在运行时间为$O(T)$的关于L的非确定性判定器

<br>

P类问题的定义(Polynomial-time problem)
<能在多项式时间内求解>
$P=\bigcup D_{time}(n^c)$
已知语言$L\subseteq\{0,1\}^*$, 称L属于P类问题
如果存在多项式时间图灵机M:
$\forall x\in \{0,1\}^*,x\in L\iff M(x)=1$

NP类问题的定义一(Non-deterministic Polynomial-time problem)
<能在多项式时间内验证, 但总能在指数时间内通过穷举求解>
已知语言$L\subseteq\{0,1\}^*$, 称L属于NP类问题
如果存在多项式时间图灵机M(验证器):
$\forall x\in \{0,1\}^*,x\in L\iff\exists u\in\{0,1\}^{p(|x|)},M(x,u)=1$
其中将u称为关于x的证明, 例如SAT问题中的u是使范式为真的一种赋值方式

NP类问题的定义二
<能通过NDTM在多项式时间内 通过穷举转移函数求解>
$NP=\bigcup ND_{time}(n^c)$

NP的两种定义是等价的
$\impliedby$: 已知语言L可被多项式时间NDTM判定
那么存在转移函数的选择序列u, 使得x进入状态$q_{accept}$
那么可构造验证器TM, 在多项式时间内沿着u来验证$M(x,u)=1$
$\implies$: 已知语言L有多项式验证器为$M(x,u)$, 现欲构造多项式NDTM
对于所有转移函数的选用序列 $u\in\{0,1\}^*$ (其中选择$\delta_0$输出0,选择$\delta_1$则输出1)
运行多项式验证器M, 如果$M(x,u)=1$, 则将此时所处状态标记为$q_{accept}$

映射归约的定义(Polynomial-time Reduction)
<语言到语言的多对一映射>
已知语言$L,L'\subseteq\{0,1\}^*$, 称$L\leq_m L'$
如果存在多项式时间可计算函数$f:\{0,1\}^*\to\{0,1\}$

传递性: $L\leq_m L'\leq_m L''\implies L\leq_m L''$
也就是说, 语言L的字符串都可在多项式时间转换后, 被L'判定

## 布尔可满足性问题(SAT)

已知布尔公式$\phi(u_1,\cdots,u_n)$
如果存在赋值$z=(z_1,\cdots,z_n)$使得$\phi(z)=1$, 
则称$\phi$为可满足的, 否则称$\phi$为不可满足的

合取范式的定义(CNF, Conjunctive Normal Form)
<多个或子句 相与>
形如$(x_{11}\lor\cdots\lor x_{1m_1})\land\cdots\land(x_{n1}\lor\cdots\lor x_{nm_n})$
其中将$(\bigvee_j x_{ij})$称为子句(clause), $x_{ij}$称为文字(literal)
如果每个子句最多包含k个文字, 则称为k-CNF, 其中可满足的称为k-SAT
$x_1=y_1\iff(x_1\lor\bar y_1)\land(\bar x_1\lor y_1)$


对于任意布尔函数$f:\{0,1\}^n\to\{0,1\}$
都存在对应的长度为$n2^n$的CNF公式, 满足$f(x)=1\iff \phi(x)=1$
证明: 由于任取$v\in\{0,1\}^n$, 都存在对应的或子句, 使得$C_v(x)=0\iff x=v$
若使得布尔函数$f(x)=0$的恰有$v_1,\cdots,v_k\in\{0,1\}^n,k\leq 2^n$
那么可构造长度至多为$n2^n$的CNF公式$\phi=C_{v_1}\land C_{v_2}\land\cdots\land C_{k}$
例如$v=\langle 1,1,0,1\rangle\to C_v=(\bar x_1\lor\bar x_2\lor x_3\lor\bar x_4)$

### 库克-勒维定理

库克-勒维定理(Cook-Levin Theorem)
1. SAT问题是NP-完全问题
2. 3-SAT问题是NP-完全问题

首先证明$\text{SAT}\in \text{NP}$
对于任意可满足的布尔表达式, 若给出一个可满足的赋值, 
那么可以在多项式时间内将该赋值代入表达式进行验证

然后证明$\forall L\in \text{NP},L\leq_m \text{SAT}$
因为$L\in\text{NP}$, 所以存在判定它的非确定性图灵机$M$
现欲找到多项式时间映射$f:L\to\text{SAT}:f(x)=\phi_x$
满足$(x\in L\iff M(x)=1)\iff\phi_x\in\text{SAT}$
即 非确定性判定格局序列$\exists Z=(z_1,\cdots,z_k)\iff\phi_x(Z)=1$

定义$\phi_x$的变量种类如下:
| 格局变量    | 解释                   | 复杂度      |
| ----------- | ---------------------- | ----------- |
| $T_{i,c,k}$ | 第k步时第i格是字符c    | $O(p(n)^2)$ |
| $H_{i,k}$   | 第k步时读写头指向第i格 | $O(p(n)^2)$ |
| $Q_{q,k}$   | 第k步时状态为q         | $O(p(n))$   |

定义$\phi_x$为以下表达式的合取:
| 约束表达式                                                                                      | 解释                     | 子句长度                   |
| ----------------------------------------------------------------------------------------------- | ------------------------ | -------------------------- |
| $T_{i,x_i,k=0}$                                                                                 | 初始输入x                | $O(p(n))$                  |
| $Q_{q_0,k=0}$                                                                                   | 初始状态$q_0$            | $O(1)$                     |
| $H_{0,k=0}$                                                                                     | 初始读写头               | $O(1)$                     |
| $\neg(T_{i,c,k}\land T_{i,c',k+1})\lor T_{i,\exists c,k}$                                       | 每格有且仅有一个元素     | $O(p(n)^2)\|\Sigma\|^2$    |
| $\neg(Q_{q,k}\land Q_{q',k})\lor Q_{\exists q,k}$                                               | 每刻有且仅有一个状态     | $O(p(n))\|Q\|^2$           |
| $\neg(H_{i,k}\land H_{i',k})\lor H_{\exists i,k}$                                               | 每刻有且仅有一个读写头   | $O(p(n)^3)$                |
| $T_{i,c,k}\lor T_{i,c',k+1}\to H_{i,k}$                                                         | 如果带被写, 说明有读写头 | $O(p(n)^2)\|\Sigma\|^2$    |
| $(H_{i,k}\land Q_{q,k}\land T_{i,c,k})$                                                         | 非确定性转移函数         | $O(p(n)^2)\|Q\|\|\Sigma\|$ |
| $\to\bigvee\limits_{((q,a),(q',c',d))\in\delta}(H_{i+d,k+1}\land Q_{q',k+1}\land T_{i,c',k+1})$ |
| $\bigvee\limits_{0\le k\le p(n)}\bigvee\limits_{h\in\text{Halt} }Q_{h,k}$                       | 在多项式时间内停机       | $O(p(n))\|Q\|$             |

如表格所示, $\phi_x$的长度最多为$O(p(n)^3)$, 故$f$是多项式时间映射
现欲证明当前构造的映射满足$\forall x\in\{0,1\}^*,(x\in L\iff M(x)=1)\iff f(x)\in\text{SAT}$
$\implies$: 若NDTM判定x, 那么说明存在某个可能的格局序列$Z=(z_1,\cdots,z_k)$使其接受并停机
用该格局序列Z给变量赋值$Y=(T_{i,c,k},H_{i,k},Q_{q,k})$, 即可得到$\phi_x(Y)=1$, 说明$\phi_x$可满足
$\impliedby$: 若$\phi_x$可满足, 那么说明存在某个可能的变量赋值方式$Y=(T_{i,c,k},H_{i,k},Q_{q,k})$使其为真
用该赋值变量Y来构造格局$Z=(z_1,\cdots,z_k)$, 即可得到$M(Z)=1$, 说明$x$可被NDTM判定

### [将SAT归约到3SAT](https://opendsa-server.cs.vt.edu/ODSA/Books/Everything/html/SAT_to_threeSAT.html)

给定任意SAT布尔公式$C_1\land C_2\land\cdots\land C_n$
那么它的或子句只会有以下四种情况:
1. 恰好包含3个文字: $(x_1+\bar x_2+x_3)$
2. 包含1个文字: $(x_1)$
3. 包含2个文字: $(x_1+\bar x_2)$
4. 包含多于3个文字: $(x_1+\bar x_2+x_3+\cdots+x_k),k>3$


#### 包含1个文字

$C=(x_1)$
$Z=(x_1+Y_1+Y_2)*(x_1+\bar Y_1+Y_2)*(x_1+Y_1+\bar Y_2)*(x_1+\bar Y_1+\bar Y_2)$
![image.png](https://s2.loli.net/2024/04/26/hwgXEa3HWKC6eGV.png =300x)

#### 包含2个文字

$C=(x_1+x_2)$
$Z=(x_1+x_2+Y_1)*(x_1+x_2+\bar Y_1)$
![image.png](https://s2.loli.net/2024/04/26/wSLnMUPTKYiQxrv.png =300x)

#### 包含多于3个文字

$C=(x_1+x_2+\cdots+x_k)$
$Z=(x_1+x_2+Y_1)*(\bar Y_1+x_3+Y_2)*(\bar Y_2+x_4+Y_3)\cdots(\bar Y_{k-3}+x_{k-1}+x_k)$
要保持相抵链的成立, 每个子句中至少要有三个文字, 故此处研究3-SAT
1. 当C恒为假时, 对于Z有$Y_1*\bar Y_1$相抵, $Y_2*\bar Y_2$相抵, $\cdots Y_{k-3}*\bar Y_{k-3}$相抵
    恰好使得Z也为假, 满足条件
2. 当C可为真时, 不妨设是$x_3=1$, 那么Z就会出现相抵链的破缺
    使得$Y_1=1$能取到, Z也为真, 满足条件
