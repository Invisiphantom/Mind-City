\phantom{+0\times} 0000   &\cdots1101\times 0(PP2)\\

[The Implementation of Booth’s Algorithm](https://www.youtube.com/watch?v=tnLKU07b-HA&list=PLBlnK6fEyqRgLLlzdgiTUKULKJPYc0A4q&index=91)
基2布斯有符号数乘法
$A=a_{n-1}a_{n-2}\cdots a_1a_0$
$B=b_{n-1}b_{n-2}\cdots b_1b_0$
$A\times B=-a_{n-1}\times B\times 2^{n-1}+\cdots+a_1\times B\times 2^1+a_0\times B\times 2^0$
$=(-a_{n-1}+a_{n-2})\times B\times 2^{n-1}+\cdots+(-a_1+a_0)\times B\times 2^1+(0-a_0)\times B\times 2^0$

| $a_ka_{k-1}$ | Operation    |
| ------------ | ------------ |
| 00           |              |
| 11           |              |
| 01           | $Acc+M$      |
| 10           | $Acc+\bar M$ |

<br>

基4布斯有符号数乘法
$A=a_{n-1}a_{n-2}\cdots a_1a_0$
$B=b_{n-1}b_{n-2}\cdots b_1b_0$
$A\times B=-a_{n-1}\times B\times 2^{n-1}+\cdots+a_1\times B\times 2^1+a_0\times B\times 2^0$
$=(-2a_{n-1}+a_{n-2}+a_{n-3})\times B\times 2^{n-2}$
$+(-2a_{n-3}+a_{n-4}+a_{n-5})\times B\times 2^{n-4}+\cdots$
$+(-2a_3+a_2+a_1)\times B\times 2^2$
$+(-2a_1+a_0+a_{-1})\times B\times 2^0$

| $a_{k+1}a_ka_{k-1}$ | Operation     |
| ------------------- | ------------- |
| 000                 |               |
| 111                 |               |
| 001                 | $Acc+M$       |
| 010                 | $Acc+M$       |
| 011                 | $Acc+2M$      |
| 101                 | $Acc+\bar M$  |
| 110                 | $Acc+\bar M$  |
| 100                 | $Acc+2\bar M$ |
