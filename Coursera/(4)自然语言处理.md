N-Gram模型
| Model    | Prob                          | Desc                       |
| -------- | ----------------------------- | -------------------------- |
| Uni-gram | P(target)                     | 只考虑单词的出现频率       |
| Bi-gram  | P(target\|context)            | 考虑相邻的两个词之间的关系 |
| Tri-gram | P(target\|context1, context2) | 考虑相邻的三个词之间的关系 |

Add-one Smoothing
P(wi|wi-1) = (C(wi-1, wi) + 1) / (C(wi-1) + V)

PRP-人称代词, VBP-动词, DT-限定词, NN-名词, TO-不定式标记, VB-动词
