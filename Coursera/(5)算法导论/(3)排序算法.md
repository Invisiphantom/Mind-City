## 示例1: 插入排序

```py
for i in range(1, len(A)):
    key = A[i] # 取出第i张牌
    k = i - 1  # 将其插入前(i-1)有序牌堆中
    while k >= 0 and A[k] > key:
        A[k + 1] = A[k]
        k = k - 1
    A[k + 1] = key
```

### 可行性分析

循环不变量: 在每次循环开始时, 子数组$\{0,\cdots,j-1\}$是有序的

起始: 第0张牌是自有序的
递推: 已知第$\{0,\cdots,j-1\}$牌有序, 由算法过程可知, 插入第j张牌后仍保持有序
结束: 当j=n时, 满足第$\{0,\cdots,n-1\}$牌有序, 即整个牌堆有序

### 最坏时间复杂度分析

最坏情况: 如果输入数组已反向排序
那么对于第j张牌, while循环需要比较j次
因此时间复杂度 $\sum_1^{n-1} j=\frac{n(n-1)}{2}=\Theta(n^2)$



## 堆排序

```py
def heapify_down(A, size, index):
    """自顶向下调整堆"""
    maxI = index
    lchdI = 2 * index + 1
    rchdI = 2 * index + 2

    # 获取当前家庭最大结点
    if lchdI < size and A[lchdI] > A[maxI]:
        maxI = lchdI
    if rchdI < size and A[rchdI] > A[maxI]:
        maxI = rchdI

    # 如果最大结点不是根结点, 需要交换
    if maxI != index:
        A[index], A[maxI] = A[maxI], A[index]  # 交换
        heapify_down(A, size, maxI)  # 递归调用


def heap_sort(A):
    n = len(A)

    # 构建最大堆
    for i in range(n // 2 - 1, -1, -1):
        heapify_down(A, n, i)

    # 逐个将最大值移到尾部, 并重新调整堆
    for i in range(n - 1, 0, -1):
        A[i], A[0] = A[0], A[i]
        heapify_down(A, i, 0)
```

不稳定
调整堆复杂度: O(logn)
建堆复杂度: O(n) TODO
总复杂度: O(nlogn)



## 快速排序


```py
def quickSort(A):
    if len(A) <= 1:
        return A
    pivot = random.choice(A)
    left = quickSort([x for x in A if x < pivot])
    right = quickSort([x for x in A if x > pivot])
    mid = [x for x in A if x == pivot]
    return left + mid + right
```
