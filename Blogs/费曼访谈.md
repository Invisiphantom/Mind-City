# 思维方式
  我怀疑每个人脑中的思考过程可能截然不同。
  大家以为他们在做同样的事，但其实他们大脑的实际运行过程却存在着巨大差异。
  当我们每个人在学习这些相同知识的时候，其实我们储存这些信息的图景和方法，以及我们思考这些知识的方式，在我们了解对方的想法后，会发现互相之间真的截然不同。

# 想象的局限 量子力学图像化

  假设小物体的行为与你所熟悉的大物体的行为非常不同。
  因为当动物进化的时候，当大脑进化的时候，大脑是习惯于处理…大脑是为普通情况而设计的。
  但如果基本例子的深层次的运行机制是由一些其他的规则和特征所决定，它们的行为方式与任何大尺度的物体都完全不同，那么在理解和想象真实时就会遇到一些困难。
  而这些困难，我们必须面对。

  所以我们知道我们的理论是正确的，但我们没有与理论相匹配的图像，也可能是因为并没有正确的图像可供人们用自己熟悉的事物取描绘。
  那么是否有可能通过学习，用手头上不熟悉的事物发展出一种熟悉度，直到这些成为第二天性。
  随着世代传承，他们能否发明教学方法使得新兴人类学会看待事物的巧妙方式，而且受过良好训练，这样他们在描述原子时就不会有我们的麻烦。

  I think Nature's imagination is so much greater than human's, she's never going to let us relax.
