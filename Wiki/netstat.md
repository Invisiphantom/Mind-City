netstat

显示活动的 TCP 连接数、计算机侦听的端口、以太网统计信息、IP 路由表、IPv4 统计信息(针对 IP、ICMP、TCP 和 UDP 协议),
以及 IPv6 统计信息(针对 IPv6、ICMPv6、TCP over IPv6 和基于 IPv6 协议的 UDP)
在不使用参数的情况下，此命令会显示活动的 TCP 连接数

------

| Option          | Description       |
| --------------- | -------------- |
| -a              | 显示所有连接    |
| -n              | 显示IP地址和端口号 |
| -o              | 显示计算机进程ID |
