man - an interface to the system reference manuals

------

| Option | Description       |
| ------ | -------------- |
| -L     | 指定语言       |
| -f     | 显示所有手册   |
| -k     | 简短搜索关键字 |
| -K     | 全局搜索关键字 |


```bash
section
    1   可执行程序或 shell 命令
    2   系统调用(内核提供的函数)
    3   库调用(程序库中的函数)
    4   特殊文件(通常位于 /dev)
    5   文件格式和规范，如 /etc/passwd
    6   游戏
    7   杂项(包括宏包和规范)
    8   系统管理命令(通常只针对 root 用户)
    9   内核例程(非标准)
```
