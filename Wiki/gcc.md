gcc - GNU project C and C++ compiler

------

| Option    | Description      |
| --------- | ---------------- |
| `-std=`   | 指定C++标准      |
| `-Wall`   | 显示所有警告     |
| `-Werror` | 将警告视为错误   |
| `-I`      | 指定头文件路径   |
| `-o`      | 指定输出文件名   |
| `-v`      | 显示编译过程     |
| `-O`      | 优化             |
| `-g`      | 生成调试信息     |
| `-E.i`    | 预处理器cpp      |
| `-S.s`    | 只编译不汇编     |
| `-c.o`    | 编译但不链接     |
| `-l`      | 指定链接库文件   |
| `-shared` | 生成动态链接库   |
| `-fPIC`   | 生成位置无关代码 |
